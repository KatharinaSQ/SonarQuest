Was ist SonarQuest?

Das Warten des Codes ist in der Regel eine unbeliebte Aufgabe bei den Entwicklern. Daher bleiben in SonarQube vermehrt Issues offen und werden nicht bearbeitet. Dies kann auf lange Sicht zu komplexen Problemen in der Software führen. Daher ist es wichtig auf eine gute Softwarequalität zu setzen.

SonarQuest ist eine Anwendung die es den Entwicklern erleichtern soll, die besagte Softwarequalität zu verbessern. Es sind dieselben Issues zu sehen wie in SonarQube auch, da SonarQuest die Daten von SonarQube bezieht. Der Unterschied ist: SonarQuest setzt auf Gamification.

Angelehnt an ein Rollenspiel werden die Entwickler im Team auf eine Abenteuerreise geschickt. Die Issues werden in Quests verpackt, sie bekämpfen Monster und schließen somit Sicherheitslücken, lösen Rätsel indem sie Tests schreiben etc. Zur Belohnung gibt es Erfahrungspunkte und Gold, womit auf dem integrierten Markt Artefakte für den eigenen Avatar gekauft werden können. Der spielerische Ansatz soll das Team motivieren, die Qualität im Projekt zu steigern

WICHTIG!

Damit die Punkte für das gelöste Issue auch gutgeschrieben werden kann muss die Quest und der Issue im Vorfeld angenommen werden!!
